# AI Recommendation System 🤖

## 📌 Project Description
This is a simple AI-based recommendation system that suggests courses/products based on user interests.

---

## ⚙️ Features
- User login system
- Interest-based recommendations
- Ranking system (match scoring)
- Saved user history
- GUI version (Tkinter)

---

## 📁 Files Included
- app.py → CLI version (terminal based)
- gui_app.py → GUI version (window app)
- users.json → stores user data
- README.md → project instructions
